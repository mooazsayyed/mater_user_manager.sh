## Introduction

Bash Friday's User Account Manager is a collection of Bash scripts designed to simplify user account management and reporting on Linux systems. This project is a part of the ongoing Bash Friday series, aimed at enhancing command-line skills and system administration proficiency.

As the IT landscape evolves towards cloud-based solutions, the ability to manage user accounts efficiently remains crucial. These scripts not only streamline user management but also provide options for generating customized user reports, making them valuable tools for both traditional and cloud-based environments.

## Features

- **User Account Management:**
  - Create User Accounts
  - Modify User Accounts
  - Delete User Accounts
  
- **User Account Reporting:**
  - Generate User Reports
  - Customizable Reports
  - Export Reports (CSV, PDF, etc.)

- **User-Friendly Interface:**
  - Clear prompts and instructions
  - Error handling and validation

## Getting Started

To get started with the Bash Friday User Account Manager, follow these steps:

1. Clone this repository to your local machine:
   ```bash
   git clone https://github.com/yourusername/bash-friday-user-manager.git
   ```

2. Make the scripts executable:
   ```bash
   chmod +x useradder.sh usermodifier2.sh userdelete.sh userreporter.sh fcustomreporter.sh
   ```

3. Run the scripts using the provided instructions in each script's section.

## Usage

The User Account Manager offers simplified usage:

- To create a new user account, run: `./useradder.sh`
- To modify an existing user account, run: `./usermodifier2.sh`
- To delete a user account, run: `./userdelete.sh`
- To generate user account reports, run: `./userreporter.sh`
- For advanced reporting options, run: `./fcustomreporter.sh`
